# Context

This database is dedicated to capturing and documenting agent skills and their host environments (extensions, plugins), serving as a reference to improve efficiency when creating future skills.

## Core Guidelines
- **Generality First:** Information must focus on universal best practices applicable to all agents whenever possible.
- **Extension & Plugin Architectures:** Broader extension and plugin architectures that host or deploy skills should be documented. Focus on universal deployment concepts where possible, but maintain distinct, clearly labeled sections for how each specific agent implements its extension/plugin system.
- **Agent-Specific Content:** Any content or examples specific to a particular agent must be clearly labeled as such. Do not present agent-specific mechanics as universal rules.
- **Target Agents (Priority Order):**
  1. Gemini (Primary focus)
  2. Codex
  3. Cursor
  4. Claude Code

When distilling information, extract the core ideas efficiently. Exact wording from sources is not required to be preserved.
